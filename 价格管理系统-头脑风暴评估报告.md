### 2.2 问题清单

#### 问题 #1：用户密码重置未加密

| 属性 | 值 |
|------|-----|
| 严重程度 | 🟠 P1 - 高危 |
| 影响范围 | 安全管理 |
| 发现位置 | `UserController.java:112-126` |
| 描述 | `resetPassword` 方法直接将明文密码存入数据库，未使用BCrypt加密 |
| 影响 | 重置的密码无法登录，因为登录时使用 `passwordEncoder.matches()` 验证 |
| 代码片段 | `user.setPassword(newPassword); // 应改为 encode(newPassword)` |
| **状态** | ✅ **已修复** - 代码第120行已使用 `passwordEncoder.encode(newPassword)` |

---

#### 问题 #2：产品分类字段映射不一致

| 属性 | 值 |
|------|-----|
| 严重程度 | 🟡 P2 - 中危 |
| 影响范围 | 产品管理 |
| 发现位置 | 前端 `ProductEdit.vue` vs 后端 `Product.java` |
| 描述 | 前端发送 `categoryId`，后端期望 `category` 对象，JPA无法自动映射 |
| 影响 | 新建/编辑产品时分类可能为null |
| **状态** | ✅ **已修复** - `ProductService.java` 第73-78行和99-105行已处理 `categoryId` 到 `category` 对象的转换 |
