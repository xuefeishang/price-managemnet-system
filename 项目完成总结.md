# 价格展示与管理系统 - 项目完成总结

## 项目概述

本项目是一个前后端分离的价格展示与管理系统，采用 Vue 3 + TypeScript 前端和 Spring Boot 3 + Java 17 后端技术栈，主要面向制造业企业内部员工使用。

## 项目位置

`E:\ClaudeCodeProject\price-management-system\`

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue 3 + TypeScript + Vant UI + Pinia + Vue Router + Axios + Vite |
| 后端 | Spring Boot 3.2.0 + Java 17 + Spring Data JPA + Spring Security + JWT |
| 数据库 | MySQL 8.4 |
| Excel处理 | EasyExcel |

## 已完成的功能

### 1. 用户认证与权限管理
- JWT Token认证
- 三种角色：管理员（ADMIN）、编辑者（EDITOR）、查看者（VIEWER）
- 登录/登出功能

### 2. 产品分类管理
- 分类列表展示
- 新增/编辑/删除分类
- 启用/禁用分类

### 3. 产品与价格管理（核心功能）
- 产品列表（支持名称、编码搜索）
- 产品详情查看
- 产品新增/编辑/删除
- 价格管理（原价、现价、成本价）
- 价格历史记录追踪

### 4. Excel导入导出
- 产品数据Excel导入
- 产品数据Excel导出

### 5. 响应式设计
- PC端布局（≥1024px）
- 移动端布局（<1024px）
- 窗口自适应调整

### 6. 操作日志管理（仅管理员）
- 操作日志列表（分页、条件筛选）
- 统计分析页面
  - 概览卡片（总操作数、登录次数、活跃用户）
  - 趋势图（ECharts 折线图）
  - 操作类型分布（ECharts 饼图）
  - 模块分布（ECharts 柱状图）
  - 用户活跃度排行
- 月度报表（按日统计 + 操作类型分布）
- 年度报表（月度趋势 + 用户年度排行金/银/铜徽章）

### 7. 数据可视化
- 首页日期选择 + 历史价格查看
- 产品详情页价格趋势折线图（ECharts）
- 日志统计多图表可视化

### 8. 审批流程管理（仅管理员）
- 审批工作流配置
  - 创建/编辑/删除工作流
  - 激活/停用工作流
  - 支持价格变更、产品创建两种工作流类型
- 审批节点配置
  - 节点类型：审批节点、知会节点
  - 审批角色：管理员、编辑者
  - 节点顺序配置
- 审批请求管理（编辑者可访问）
  - 待我审批列表
  - 我的申请列表
  - 审批通过/拒绝/撤回

### 9. 字典管理
- 数据字典 CRUD（按分类管理，支持启用/停用）
- 前端字典服务（useDict composable，全局缓存 + 便捷方法）
- 所有页面硬编码编码标签替换为字典服务动态获取
  - 状态标签（ACTIVE/INACTIVE → getStatusLabel）
  - 角色选项/名称（→ getRoleLabel/getDictOptions）
  - 货币符号（→ getCurrencySymbol）
  - 操作类型/模块/变更类型等（→ getDictValue）
- 字典管理页面（DictManagement.vue，按分类筛选和编辑）
- 预置字典分类：币种、通用状态、用户角色、审批状态、工作流类型、节点类型、业务类型、审批操作、变更类型、计量单位、操作类型、操作模块、菜单图标

## 快速启动

### 1. 初始化数据库

```sql
CREATE DATABASE IF NOT EXISTS price_management DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE price_management;
source E:\ClaudeCodeProject\price-management-system\backend\src\main\resources\init.sql;
```

### 2. 启动后端

使用 IntelliJ IDEA 打开 `backend` 目录，运行 `PriceManagementApplication.java`

或命令行：
```bash
cd backend
mvnw spring-boot:run
```

### 3. 启动前端

```bash
cd frontend
npm install
npm run dev
```

### 4. 访问应用

打开浏览器访问：http://localhost:5173
