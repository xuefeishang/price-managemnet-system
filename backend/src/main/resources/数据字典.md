## 表结构详情

### sys_user - 系统用户表

**说明**：存储系统用户信息，支持三种角色（管理员、编辑者、查看者）

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | - | 用户ID |
| username | VARCHAR(50) | NOT NULL, UNIQUE | - | 用户名 |
| password | VARCHAR(200) | NOT NULL | - | 密码（BCrypt加密） |
| role | VARCHAR(20) | NOT NULL | - | 角色：ADMIN/EDITOR/VIEWER |
| status | VARCHAR(20) | NOT NULL | 'ACTIVE' | 状态：ACTIVE/INACTIVE |
| nickname | VARCHAR(50) | - | - | 昵称 |
| email | VARCHAR(100) | - | - | 邮箱 |
| phone | VARCHAR(20) | - | - | 电话 |
| created_time | DATETIME | NOT NULL | CURRENT_TIMESTAMP | 创建时间 |
| updated_time | DATETIME | - | CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**：
- `idx_user_username` (username)
- `idx_user_status` (status)
- `idx_user_role` (role)

**枚举值**：

`role` 字段：
| 值 | 说明 |
|----|------|
| ADMIN | 管理员 - 拥有所有权限 |
| EDITOR | 编辑者 - 可编辑产品和价格 |
| VIEWER | 查看者 - 仅可查看数据 |

`status` 字段：
| 值 | 说明 |
|----|------|
| ACTIVE | 启用状态 |
| INACTIVE | 禁用状态 |

### product_category - 产品分类表

**说明**：存储产品分类信息，支持树形结构扩展

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | - | 分类ID |
| name | VARCHAR(100) | NOT NULL | - | 分类名称 |
| code | VARCHAR(50) | UNIQUE | - | 分类编码 |
| sort_order | INT | - | 0 | 排序顺序 |
| status | VARCHAR(20) | NOT NULL | 'ACTIVE' | 状态：ACTIVE/INACTIVE |
| remark | TEXT | - | - | 备注 |
| created_time | DATETIME | NOT NULL | CURRENT_TIMESTAMP | 创建时间 |
| updated_time | DATETIME | - | CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**：
- `idx_category_code` (code)
- `idx_category_status` (status)
- `idx_category_sort` (sort_order)

**初始数据**：
| name | code | sort_order |
|------|------|------------|
| 黑色金属 | BLACK_METAL | 1 |
| 有色金属 | NON_FERROUS_METAL | 2 |
| 贵金属 | PRECIOUS_METAL | 3 |
| 化工产品 | CHEMICAL | 4 |
| 煤炭及焦炭 | COAL | 5 |

### product - 产品表

**说明**：存储产品基本信息，与分类表多对一关联

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | - | 产品ID |
| name | VARCHAR(200) | NOT NULL | - | 产品名称 |
| selling_price | DECIMAL(15,4) | - | - | 售价 |
| category_id | BIGINT | FOREIGN KEY | - | 分类ID |
| status | VARCHAR(20) | NOT NULL | 'ACTIVE' | 状态：ACTIVE/INACTIVE |
| description | TEXT | - | - | 产品描述 |
| specs | TEXT | - | - | 产品规格 |
| image_url | VARCHAR(500) | - | - | 图片URL |
| remark | TEXT | - | - | 备注 |
| unit | VARCHAR(50) | - | - | 计量单位 |
| sort_order | INT | - | 0 | 排序顺序 |
| created_time | DATETIME | NOT NULL | CURRENT_TIMESTAMP | 创建时间 |
| updated_time | DATETIME | - | CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**外键**：
- `category_id` → `product_category(id)` ON DELETE SET NULL ON UPDATE CASCADE

**索引**：
- `idx_product_category` (category_id)
- `idx_product_status` (status)
- `idx_product_name` (name)
- `idx_product_sort` (sort_order)

**枚举值**：

`unit` 字段：
| 值 | 说明 |
|----|------|
| 元/吨 | 每吨价格（元） |
| 万元/吨 | 每吨价格（万元） |
| 元/克 | 每克价格（元） |
| 元/千克 | 每千克价格（元） |

`status` 字段：
| 值 | 说明 |
|----|------|
| ACTIVE | 启用状态 |
| INACTIVE | 禁用状态 |

### price - 价格表

**说明**：存储产品价格信息，支持价格有效期管理

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | - | 价格ID |
| product_id | BIGINT | NOT NULL, FOREIGN KEY | - | 产品ID |
| original_price | DECIMAL(15,4) | - | - | 原价 |
| current_price | DECIMAL(15,4) | NOT NULL | - | 现价 |
| cost_price | DECIMAL(15,4) | - | - | 成本价 |
| effective_date | DATE | - | - | 生效日期 |
| expiry_date | DATE | - | - | 失效日期 |
| unit | VARCHAR(50) | - | - | 单位：元/吨、元/克等 |
| price_spec | VARCHAR(200) | - | - | 价格规格 |
| created_by | BIGINT | - | - | 创建人 |
| created_time | DATETIME | NOT NULL | CURRENT_TIMESTAMP | 创建时间 |

**外键**：
- `product_id` → `product(id)` ON DELETE CASCADE ON UPDATE CASCADE

**索引**：
- `idx_price_product` (product_id)
- `idx_price_effective` (effective_date, expiry_date)
- `idx_price_created` (created_time)

**业务规则**：
- 同一产品的价格按生效日期分段有效
- 新价格生效时，旧价格自动设置失效日期

### price_history - 价格历史表

**说明**：记录所有价格变动历史，用于追溯和审计

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | - | 历史记录ID |
| price_id | BIGINT | FOREIGN KEY | - | 价格ID |
| product_id | BIGINT | NOT NULL, FOREIGN KEY | - | 产品ID |
| old_price | DECIMAL(15,4) | - | - | 旧价格 |
| new_price | DECIMAL(15,4) | NOT NULL | - | 新价格 |
| change_type | VARCHAR(20) | NOT NULL | - | 变动类型 |
| remark | TEXT | - | - | 备注 |
| changed_time | DATETIME | NOT NULL | CURRENT_TIMESTAMP | 变动时间 |

**外键**：
- `price_id` → `price(id)` ON DELETE SET NULL ON UPDATE CASCADE
- `product_id` → `product(id)` ON DELETE CASCADE ON UPDATE CASCADE

**索引**：
- `idx_history_product` (product_id)
- `idx_history_price` (price_id)
- `idx_history_time` (changed_time)
- `idx_history_type` (change_type)

**枚举值**：

`change_type` 字段：
| 值 | 说明 |
|----|------|
| CREATE | 新建价格 |
| UPDATE | 更新价格 |
| DELETE | 删除价格 |

### approval_request - 审批请求表

**说明**：记录每个具体的审批请求实例

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|----------|------|--------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | - | 请求ID |
| workflow_id | BIGINT | NOT NULL, FOREIGN KEY | - | 流程ID |
| business_type | VARCHAR(50) | NOT NULL | - | 业务类型 |

| current_node_id | BIGINT | FOREIGN KEY | - | 当前节点ID |
| status | VARCHAR(20) | NOT NULL | 'PENDING' | 状态 |

**外键**：

- `idx_request_business` (business_type, business_id)

**枚举值**：

`business_type` 字段：
| 值 | 说明 |
|----|------|
| PRICE | 价格相关业务 |
| PRODUCT | 产品相关业务 |

`status` 字段：
| 值 | 说明 |
|----|------|
| PENDING | 待审批 |
| APPROVED | 已通过 |

| CANCELLED | 已撤回 |

### C. 索引说明

索引用于加速查询，合理的索引设计可以显著提升查询性能。
系统中的索引设计遵循以下原则：
- 外键字段建立索引加速关联查询
- 经常查询的字段建立索引
- 唯一字段建立唯一索引
- 排序字段建立索引加速排序
